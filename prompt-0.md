Создай полноценный Python-проект под названием "AI Client Report Generator".

Задача проекта:
Автоматически формировать PDF-отчёты по диалогам с клиентами, используя ИИ и HTML-шаблоны.

🔧 Технические требования:

    Используемые библиотеки:
        openai
        jinja2 — для HTML-шаблонов
        weasyprint — для генерации PDF из HTML
        flask — для API-запросов (опционально)
        python-dotenv — для хранения API ключа в .env

    Архитектура проекта:
    project_root/
    ├── main.py # основной файл
    ├── templates/
    │ ├── report_template.html # HTML-шаблон отчёта
    ├── reports/
    │ └── (сюда сохраняются готовые PDF)
    ├── utils/
    │ ├── ai_processor.py # взаимодействие с моделью OpenAI
    │ ├── pdf_generator.py # функции HTML→PDF
    ├── .env # API ключ
    ├── requirements.txt # зависимости

    Логика работы программы:

    1️⃣ Пользователь отправляет транскрибацию диалога (текстовый файл или ввод вручную).

    2️⃣ Код вызывает функцию process_dialog_with_ai(text) из ai_processor.py, которая:

    - Передаёт транскрибацию в OpenAI.
    - Возвращает структурированные данные в JSON:
    {
    "client_name": "...",
    "topic": "...",
    "main_request": "...",
    "mood": "...",
    "next_steps": "..."
    }

    3️⃣ Далее pdf_generator.py:

    - Подставляет эти данные в HTML-шаблон через Jinja2.
    - Конвертирует HTML в PDF.
    - Сохраняет PDF в папку `/reports` с именем вроде `report_2025-11-13_15-30.pdf`

    4️⃣ В main.py всё объединяется:

    - Считываем транскрибацию.
    - Отправляем в ИИ.
    - Генерируем PDF.
    - Печатаем “Отчёт успешно создан: reports/report_….pdf”.
